# Amethyst 

Amethyst é um reprodutor de música cross-platform baseado em [Electron](https://electronjs.org/) com um sistema de [fluxo de sinal](https://en.wikipedia.org/wiki/Audio_signal_flow) [em nódulos](https://en.wikipedia.org/wiki/Node_graph_architecture). O objetivo principal deste projeto é criar um [reprodutor de música](https://en.wikipedia.org/wiki/Media_player_software) em [TypeScript](https://www.typescriptlang.org/) para alcançar os limites da linguagem e provar que é possivel entregar recursos de nível profissional como as de [DAWs](https://en.wikipedia.org/wiki/Digital_audio_workstation) / [DAEs](https://en.wikipedia.org/wiki/Audio_editing_software), providenciando ferramentas úteis e customizações para o [usuário](https://en.wikipedia.org/wiki/End_user).

## 📝 Contribuir
- Utilize `Node.js v18.0.0`
- Utilize as `extensões recomendadas`
- Instale `libvips` no Linux
- Instale `yarn` 

## ⌨️ Programar
- Instale as dependências com `yarn`
- Desenvolva utilizando `yarn dev`
- Compile com `yarn package`, arquívos compilados aparecem na pasta `release/build`
